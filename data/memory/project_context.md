# Project Context

Proyecto: The Best Burger WhatsApp Bot

Objetivo:
Crear un primer filtro de neuroventa para atender clientes por WhatsApp,
mostrar combos, tomar pedidos simples y escalar a Nelson solo cuando haga falta.

Rol del bot:
- vender
- reducir fricción
- ahorrar tiempo operativo viernes y sábado

Este bot es una pieza táctica validable dentro de un sistema mayor de automatización y gestión operativa.
